Always watching. Always reporting.

Vigil is an automated web monitoring and auditing platform that continuously tracks the health of any website across four critical dimensions — performance, SEO, accessibility (ADA/WCAG compliance), and security.

Built in 36 hours at HackYSU 2026.

Features:

Automated scheduling — set audits to run every 6 hours, daily, or weekly
Four-dimension scoring — performance, SEO, accessibility, and security each scored out of 100
Trend dashboard — visualize how scores change over time across every audit
Instant alerts — email notification the moment any score drops below your threshold
AI summaries — plain-English explanations generated by Claude (Anthropic API), written for clients and executives
ADA/WCAG compliance checking — detects the specific issues that create legal liability
Multi-site management — monitor any number of URLs simultaneously
Zero manual effort — set it once, let it run
